tag utilização
<head> Contém informações sobre o documento que não são exibidas diretamente, como metadados, título e links para recursos externos.
<header> Elemento semântico que representa o cabeçalho de uma página ou seção, geralmente contendo navegação e identidade visual.
<title> Define o título do documento, exibido na aba do navegador e utilizado por mecanismos de busca.
<nav> Elemento semântico destinado a agrupar links de navegação principais, como menus e sumários.
<main> Representa o conteúdo principal e único da página, excluindo cabeçalhos, rodapés e barras laterais.
<section> Representa uma seção temática do documento. Por convenção, deve sempre conter um elemento de título.
<div> Contêiner genérico sem valor semântico, amplamente utilizado para agrupar elementos e facilitar a estilização via CSS.
<li> Representa um item de lista, podendo ser filho de ul (lista não-ordenada) ou ol (lista ordenada).
<ul> Define uma lista não-ordenada, onde a sequência dos itens não implica ordem ou prioridade.
<a> Cria hiperlinks para outras páginas, arquivos, endereços de e-mail ou seções do mesmo documento através do atributo href.
<p> Define um parágrafo de texto, adicionando automaticamente espaçamento antes e depois do conteúdo.
<span> Contêiner inline genérico sem valor semântico, utilizado para aplicar estilos a trechos específicos de texto.
<h1>, <h2>...<h6> Definem títulos com seis níveis de hierarquia, onde h1 é o mais importante.
<img> Incorpora uma imagem ao documento. Os atributos src (caminho) e alt (descrição) são essenciais para acessibilidade.
<footer> Elemento semântico que representa o rodapé de uma página ou seção, geralmente com informações de contato e créditos.
